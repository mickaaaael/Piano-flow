Prêt à jouer ?

Les notes descendent du haut. Quand une barre touche le clavier, c'est le moment de jouer cette touche.

À la souris : utilisez la molette sur l'écran ou la barre du bas (cliquer-glisser pour boucler).

💡 Ruban LED ESP32

Entre l'adresse IP de ton ESP32 (affichée dans le Moniteur série d'Arduino). Les LEDs s'allumeront en cyan pour la main droite et doré pour la main gauche, et les deux extrémités s'allument pour la pédale.

🎯 Calibrer le ruban LED

La LED de la touche affichée s'allume automatiquement : compare avec ton piano, ajuste avec ◀ LED / LED ▶ puis ✓ Valider — la touche suivante s'enchaîne toute seule. Sur le piano ci-dessous : vert = calibrée, cyan = en cours. Tape une touche pour y sauter. Les touches non calibrées sont ajustées par interpolation entre les points voisins.

☁ Synchronisation multi-appareils

Tes morceaux MIDI seront sauvegardés dans un GitHub Gist privé et accessibles depuis n'importe quel appareil. Il te faut un token GitHub avec la permission gist uniquement.

Créer un token : github.com → Paramètres → Paramètres développeur → Jetons d'accès personnels → Jetons classiques → cocher gist.
